DF-SDD: Distributed Feature-based Surface Defect Detection

A lightweight yet powerful surface defect detection network that introduces Distributed Convolution (DConv) and Distributed Depthwise Convolution (DDWConv) to efficiently capture both local and long-range features, while using Spatial Attention Gate (SAG) for feature fusion.

üìå Architecture Overview

Overview of DF-SDD Network

DConv and DWConv

üß© Motivation

Traditional surface defect detectors rely heavily on standard convolution, limiting their ability to handle variations in defect texture and scale. DF-SDD aims to overcome these challenges with a more flexible and efficient design suited for real-world industrial applications.

üöÄ Project Contributions

  • ‚öôÔ∏è Introduced DConv and DDWConv for larger receptive fields at low cost.
  • üß† Designed DFEB block to capture both local and global features in parallel.
  • üåê Used Spatial Attention Gate (SAG) for multi-level feature fusion across scales.
  • üìâ Achieved state-of-the-art results with only 0.85M parameters and 6.2 GFLOPs.

üõ†Ô∏è Key Features

  • üîç Accurate detection of small and large defects (scratches, pits, inclusions).
  • üìä Multi-scale feature integration with dual-path fusion (top-down & bottom-up).
  • üí° Ablation study confirms importance of SAG, DML, LRB, and LEB modules.
  • üìà Highest mAP on NEU-DET (0.894) and GC10-DET (0.725).

üî¨ System Design

The DF-SDD network consists of three main stages: feature extraction (DFEB), feature fusion (SAG-based), and prediction. It uses a decoupled head for final bounding box prediction across three scales.

  • Framework: PyTorch
  • Datasets: NEU-DET, GC10-DET, CarDD
  • Programming: Python

Specular Detection and Rendering for Immersive Multimedia

A novel deep learning framework that detects and refines specular regions in 360¬∞ video, improving immersive video quality for multimedia applications such as VR and AR.

üìå System Overview Diagram

VAUnet Architecture.

üß© Background

Specular highlights in immersive 360¬∞ content reduce visual quality and realism. Existing solutions often treat them as noise, ignoring their physical and perceptual importance. This project addresses this challenge by detecting specular regions and applying refined rendering to preserve or remove them based on context.

üöÄ Project Contributions

  • üìå Designed VAUnet: high-performance variational attention U-network for specular highlight detection.
  • üîç Hybrid loss function including combined structural loss (Dice + Cross-Entropy) with KL divergence to preserve fine boundary details and semantic coherence.
  • üé® Integrated detected masks into MPEG-I based rendering pipeline to reconstruct or suppress specular highlights contextually.
  • üìà Demonstrated robustness across synthetic and real 360¬∞ datasets .

üõ†Ô∏è Key Features

  • üß† Mapping the input image into a standard normal distribution to accurately interpolate the output.
  • üß© Attention fusion module for context-aware feature aggregation.
  • üéØ Integration with MPEG-I rendering for higher subjective rendering quality..

üî¨ System Design

The network follows an encoder-decoder structure with a standard normal distribution to encode input image and attention blocks. It uses U-net as backbone and outputs binary masks to guide rendering. The rendering system enhances specular realism or suppresses unwanted artifacts in immersive 360¬∞ content.

Deep learning based specular highlight detection and removal (SHDRN)

A two-stage network for accurate detection and removal of complex specular highlights in natural images. Combines VAUnet for precise detection with QformerSHR, a Quaternion transformer-based removal network.

üìå Architecture Overview

Two-stage SHDRN framework: VAUnet + QformerSHR

DSQConv and Quaternion Multi-head attention

üß© Motivation

Specular highlights hinder many computer vision tasks by obscuring texture and shape information. Traditional CNN-based methods fail to capture global context effectively and ignore RGB structural correlations. SHDRN addresses these issues by combining detection and removal into a unified, robust framework.

üöÄ Project Contributions

  • üéØ Proposed a two-stage SHDRN pipeline for specular highlight removal.
  • üß† Developed QformerSHR using Quaternion representation for RGB structural preservation.
  • ‚ö° Introduced DSQConv for efficient computation with enhanced spatial correlation.
  • üåç Integrated QTB to capture long-range dependencies for better restoration.
  • üìà Achieved superior PSNR and SSIM compared to state-of-the-art methods.

üõ†Ô∏è Key Features

  • üîç Accurate specular mask generation with VAUnet.
  • üí° Transformer-based highlight removal for global context learning.
  • üåÄ Quaternion convolution to retain RGB inter-channel correlations.
  • üìä Hybrid training strategy for robust and efficient learning.

üî¨ System Design

SHDRN first uses VAUnet to detect and localize specular highlights, generating binary masks. These masks, along with the original image, are fed into QformerSHR, which performs highlight removal using DSQConv and QTB modules. The hybrid training strategy jointly optimizes both stages for better synergy.

üß™ Technologies Used

  • Framework: PyTorch
  • Programming: Python
  • Techniques: Quaternion Transformers, DSQConv, Image Restoration
  • Datasets: Multiple natural and text-image datasets with specular highlights

Lightweight 3D CNN for Brain MRI Super-Resolution

A computationally efficient deep learning network for enhancing brain MRI resolution, designed with 3D convolution to handle volumetric medical data. The network integrates extraction, compression, and reconstruction parts with skip connections for optimal performance while maintaining low complexity for real-time applications.

üìå Architecture Overview

Overview of the lightweight 3D CNN architecture

üß© Motivation

High-resolution brain MRI provides exceptional anatomical detail but is often limited by long scan times, high magnetic field requirements, and low SNR. Conventional super-resolution networks are computationally heavy and unsuitable for integration into MR scanners. This project aims to design an efficient, high-quality MRI SR network that meets real-time constraints.

üöÄ Project Contributions

  • ‚öôÔ∏è Designed a lightweight 3D CNN tailored for volumetric brain MRI data.
  • üîó Integrated skip connections between extraction, compression, and reconstruction modules for better gradient flow.
  • üìâ Reduced computational cost using 1√ó1√ó1 convolution in the compression stage.
  • üìä Achieved high PSNR and SSIM with significantly fewer parameters than conventional networks.

üõ†Ô∏è Key Features

  • üß† Handles volumetric MRI data directly with 3D convolutions.
  • üí° Efficient network design suitable for integration into MR scanners.
  • üñºÔ∏è Produces clearer anatomical details without over-smoothing.
  • ‚ö° Fast inference speed for clinical applications.

üî¨ System Design

The network consists of:

  • Extraction part: Captures low-level 3D features from MRI volumes.
  • Compression part: Reduces redundancy and complexity using 1√ó1√ó1 convolutions.
  • Reconstruction part: Restores high-resolution MRI with preserved anatomical details.
Skip connections are employed to preserve spatial information across stages.

üß™ Technologies Used

  • Framework: PyTorch
  • Programming: Python
  • Domain: Medical Image Processing, MRI
  • Model Type: 3D Convolutional Neural Network
